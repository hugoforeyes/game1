# Player Sprites

Place your player spritesheet here as `player.png`.

Recommended: 16x16 or 32x32 per frame, 4-direction walk cycle (down/up/left/right).
Each direction: 4 frames walk + 1 idle frame.

Example layout (16x16 tiles, 4 cols x 5 rows):
- Row 0: walk_down  (4 frames)
- Row 1: walk_up    (4 frames)
- Row 2: walk_left  (4 frames)
- Row 3: walk_right (4 frames)
- Row 4: idle       (1 frame each direction, cols 0-3)
